## AI 占星阁 · 正式版 MVP

一个基于 `Next.js` 构建的占卜网页应用 MVP，当前包含：
- 八字：农历出生信息、出生地自动换算经纬度、24 小时制时间、流年流月干支与时间轴解读
- 塔罗：单张 / 三张 / 四张 / 五张牌阵、抽牌动画、结果解读

## 本地运行

```bash
npm install
npm run dev
```

打开 `http://127.0.0.1:4173` 即可访问。

## 当前结构

- `app/`：页面与 API 路由
- `components/`：页面组件
- `lib/`：八字 / 塔罗 / 地理编码逻辑

## 说明

当前版本已从演示页迁移为正式工程结构；八字干支换算已使用 `lunar-javascript`，地理编码通过服务端接口统一处理并带本地兜底。
